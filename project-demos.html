Spam message classifier

I built a CNN plus GloVe spam filter for coursework. On the UCI test set it reached about 98.74% accuracy, 95.21% spam F1, and 99.42% ROC-AUC. The playground below uses hand-tuned rules in your browser (instant, no server), not the trained network.

For recruiters: this page is a fast browser MVP (rules-based scoring). The trained model, metrics, and limitations are on About the model. Real inboxes are harder than UCI; the CNN still misses some spam (see false negatives in outputs/test_predictions_dl.csv in the project repo).
