Spam message classifier

This tool checks if a message is spam or not. Type in any message and it tells you within seconds.

How to use it

Type a message and click Analyze. The result comes from an AI model running on a backend server - not simple rules in your browser.

What's powering it

Python, TensorFlow/Keras (AI libraries), and a small local server. Trained on real SMS data - ~98.7% accuracy on test data.

This is a learning project, not a production spam filter. For a full technical breakdown, see the model write-up page.

Common Misconceptions & Disclaimer

01

Spam is not the same as scam

Spam means automated or mass-generated text sent to large numbers of people, not that the message is fraudulent or dangerous. Your bank's two-factor code, an Amazon delivery update, or a flight reminder can all score as spam because they are automated and sent in bulk. That does not make them untrustworthy. This model is trained to detect automation patterns, not malicious intent.

02

A high score does not mean danger

The spam probability reflects how closely a message resembles the automated text patterns the model was trained on, not how harmful it is. A marketing email from a brand you signed up to may score very high simply because it uses the same language patterns as bulk spam. Likewise, a low score is not a safety guarantee. Always use your own judgement alongside any automated tool.
