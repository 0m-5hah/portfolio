Spam message classifier

This tool checks if a message is spam or not. Type in any message and it tells you within seconds.

How to use it

Type a message and click Analyze. The result comes from an AI model running on a backend server - not simple rules in your browser.

How it was built

  • Model design and comparisons
  • Examples where it gets it wrong
  • Accuracy stats and outputs

What's powering it

Python, TensorFlow/Keras (AI libraries), and a small local server. Trained on real SMS data - ~98.7% accuracy on test data.

This is a learning project, not a production spam filter. For a full technical breakdown, see the model write-up page.
